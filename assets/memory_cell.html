Single Bit

0 represents 0
1 represents 1

Two Bits

0 0 represents 0
0 1 represents 1
1 0 represents 2
1 1 represents 3

Three Bits

0 0 0 represents 0
0 0 1 represents 1
0 1 0 represents 2
0 1 1 represents 3
1 0 0 represents 4
1 0 1 represents 5
1 1 0 represents 6
1 1 1 represents 7

One Byte (8 Bits)

0 0 0 0 0 0 0 0 represents 0 (all bits 0)

32-Bit Memory Block (for int, number 4)
